# Tic-Tac-Toe

A simple command-line Tic-Tac-Toe game written in Python.

## Getting Started

### Requirements

- Python 3.7+

### Running the game

```bash
python -m src.tic_tac_toe
```

### Playing

Follow the prompts to make moves. The board positions are numbered 1 through 9:
```
1 | 2 | 3
---------
4 | 5 | 6
---------
7 | 8 | 9
```
